Requerimientos:
SQL Server 2012 (Preferiblemente 2014)
Visual Studio 2012 (Preferiblemente 2015)

Instrucciones:
1. Cree una base de datos llamada 'cinequilla' en una instancia SQL Server corriendo en el mismo PC donde se va a correr la aplicacion web.
2. Importe el archivo SQL en la base de datos recien creada
3. Ajuste los credenciales necesarios en la ConnectionString 'CineQuilla' el archivo Web.config de la aplicacion.
4. Corra la aplicacion web.
